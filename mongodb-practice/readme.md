<h1>MongoDB</h1>

1. mongodb is a <b>NOSQL</b> database
2. it stores data in <b>BSON format</b>
3. A <b>Document</b> is a file that contain the indivisual information about an object

    name:"Muhammad affan',
    age:20,

<br/>
